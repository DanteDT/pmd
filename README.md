## Power literature
Moby-Dick annotated epub, based on the great efforts of [Power Moby Dick](http://www.powermobydick.com/)
With front/back material from a [scanned, signed first-edition provided by the Internet Archive](https://archive.org/details/mobydickorwhale01melv/page/n7/mode/2up).

### Very brief overview
1. Scrape PMD TOC and Chapters
2. Basic HTML clean up, then some deep cleaning and patching to improve the e-reader experience
3. Build the EPUB

### Contribute
Suggestions welcome to improve the epub experience, or to further clean up the content.
